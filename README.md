## Installation
1. **Clone the repo**
```sh
git clone https://github.com/heykush/Tiktok
```
2. Install packages
 **Must to install**
 Copy the below commands and Run in `bash/cmd` terminal one at a time. 
 ~~~
 pip install selenium
 pip install pyautogui
 pip install time
~~~
